Detailed Description

ZLib compression class using std::string for input and output.

Implements zlib compression using std::string for input and output. To use, simply call compress() zero or more times and then call finish(). This class implements RFC 1950.

std::string zlibcomplete::ZLibCompressor::compress ( const std::string &  input)

Compresses data. Uses std::string for input and output.

Accepts a std::string of any size containing data to compress. Returns as much zlib compressed data as possible. In auto_flush mode, this will ensure that the returned data is immediately decompressible to the original data. In no_flush mode, the data may not all be available until the rest of the compressed data is written using more compress() or a finish() call. After calling compress() zero or more times, it is necessary to call finish() exactly once.

Writes a termination block to the zlib stream indicating the end.

Flushes any remaining data that has not yet been written and then writes a terminal block to the compressed data stream. Returns this compressed data as a std::string. This function should be called exactly once for each compressed data stream.
